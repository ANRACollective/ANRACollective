# ANRA Collective — A Pop-Up Portfolio

A pop-up book you can scroll. The cover opens, pages turn, and each spread
raises a paper-engineered pop-up for one project.

**The whole website is one file: [`index.html`](index.html).**
Open it in any browser — no install, no build step.

## Publish it (free, no Vercel needed)

GitHub Pages serves this repo as a website:

1. Push this repo to GitHub.
2. On GitHub: **Settings → Pages → Source: Deploy from a branch → `main` / root → Save.**
3. Your book goes live at `https://anracollective.github.io/ANRACollective/` a minute later.

## What's in here

- `index.html` — the entire website (design, story, pop-ups, mobile edition).
- `nextjs-version/` — the same site as a full Next.js project, kept for when
  we move to Vercel with a custom domain later. Ignore it for now.
